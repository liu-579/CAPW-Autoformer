# Tourism Review Multi-dimensional Sentiment Dataset V1

## 基本信息

- **版本**: v1.0
- **创建日期**: 2025-11-28
- **数据来源**: 携程旅游评论数据
- **语言**: 中文
- **领域**: 旅游评论

## 数据规模

| 集合 | 样本数 | 占比 |
|------|--------|------|
| 训练集 | 2109 | 80.0% |
| 验证集 | 264 | 10.0% |
| 测试集 | 264 | 10.0% |
| **总计** | **2637** | **100%** |

## 数据字段

| 字段名 | 类型 | 说明 | 取值范围 |
|--------|------|------|----------|
| id | string | 评论唯一标识符 | - |
| text | string | 清洗后的评论文本 | - |
| 景色 | int | 景色维度情感分数 | -2 到 2 |
| 交通 | int | 交通维度情感分数 | -2 到 2 |
| 美食 | int | 美食维度情感分数 | -2 到 2 |
| 门票 | int | 门票维度情感分数 | -2 到 2 |
| 服务 | int | 服务维度情感分数 | -2 到 2 |

## 情感评分说明

评分采用5级量表：

- **-2**: 极差（明确表达强烈不满）
- **-1**: 差（表达不满、体验不佳）
- **0**: 中性或未提及
- **1**: 好（表达满意、体验良好）
- **2**: 极好（表达强烈满意、极力推荐）

## 数据处理流程

1. 模块1：文本清洗与标准化
2. 模块2：多维情感评分（GPT-4o-mini）
3. 模块3：数据对齐、合并与划分

## 划分方法

- **随机种子**: 42（保证可复现）
- **训练集比例**: 80%
- **验证集比例**: 10%
- **测试集比例**: 10%
- **打乱数据**: 是

## 已知偏差与局限性

### 可能偏差

- 数据来源于携程平台，可能存在平台偏差
- 情感评分由GPT模型生成，可能存在模型偏差
- 评论可能更多来自对景点有强烈情感的用户

### 局限性

- 仅适用于中文旅游领域评论
- 情感维度固定为5维，可能无法完全覆盖所有情感方面
- 评分范围为-2到2，粒度有限

## 使用建议

- **适用场景**: 旅游评论多维情感分析模型训练
- **不适用场景**: 非旅游领域文本、非中文文本、其他情感维度需求
